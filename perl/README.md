# App::Aux

Developer tools for daily use.

# Building

 requires Perl 5.8.8 or higher.  Perl 5.8.8 was released January 2006.

    # Required
    perl Makefile.PL
    make
    make test
    sudo make install # for a system-wide installation (recommended)
    # - or -
    make appaux
    cp bin ~/appaux # for a personal installation
